# Customer Churn Prediction & Insights (InsightLab - Module 1)

A full-stack analytics + machine learning app to:
- analyze churn drivers
- predict churn probability for customers
- explain predictions with feature importance (later: SHAP)
- provide retention recommendations

## Tech Stack
- Backend: FastAPI (Python)
- ML: scikit-learn (later: XGBoost, SHAP)
- Frontend: React + Vite + Tailwind
- Deployment (later): Vercel (FE) + Render/Fly.io (BE)

## Repo Structure
- backend/: FastAPI API + ML inference
- frontend/: React UI
- notebooks/: EDA + model training
- models/: saved model artifacts
- data/: raw/processed data (excluded from git by default)
